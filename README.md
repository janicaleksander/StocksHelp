# StocksMaster

StocksMaster is a stock market simulator. 

## Technologies

- **Backend:** Golang
- **Styling:** Tailwind CSS
- **Performance Optimization:** htmx
- **Page Rendering:** Templ
- **Database:** PostgreSQL

### Run application:
```sh
https://stocksmaster.onrender.com/home
```
Due to free hosting, it takes a while to load.
